# Selenium TestNG Automation Project

This project is an automated UI testing framework using:

- **Java**
- **Selenium WebDriver**
- **TestNG**
- **Maven**
- **WebDriverManager**

---

## Features

- Automated browser testing using Chrome
- Test execution managed with TestNG
- HTML reports generated after tests
- Dependency management via Maven

---

## Project Structure
**src/test/java/com/automation
pom.xml
testng.xml
test-output/**

- `src/test/java/com/automation` → Your test classes  
- `pom.xml` → Maven dependencies  
- `testng.xml` → TestNG suite configuration  
- `test-output/` → Generated reports after test execution  

---

## How to Run

1. Clone the repository: git clone (https://github.com/Shivaninagula11/Selenium-TestNG-Automation)
2. 2. Open the project in Eclipse as a **Maven Project**  
3. Make sure all dependencies are downloaded  
4. Right-click on `testng.xml` → **Run As → TestNG Suite**  
5. Test results will appear in `test-output` folder
